# 功能添加指南

## 1. 添加新插件

**位置**：`plugins/` 目录

**步骤**：
1. 在 `plugins/` 目录下创建新的 Python 文件（如 `xin_功能.py`）
2. 或创建新的插件目录（如 `xin_功能/`），并在其中创建 `__init__.py` 文件
3. 按照 NoneBot 插件格式编写插件代码

## 2. 添加新的事件处理函数

**位置**：`bot.py` 文件中的 `if __name__ == "__main__":` 部分

**步骤**：
1. 在 `bot.py` 中定义新的异步函数（如 `async def on_new_event()`）
2. 在主入口部分使用 `driver.on_事件类型(处理函数)` 注册事件

## 3. 修改上线消息内容

**位置**：`fa song_shangxian_message` 函数

**步骤**：
1. 修改 `await bot.send_private_msg()` 中的 `message` 参数值

## 4. 添加新的配置项

**步骤**：
1. 在项目根目录创建或修改 `.env` 文件
2. 添加新的配置项（如 `NEW_CONFIG=value`）
3. 在代码中使用 `driver.config.new_config` 获取配置值

## 5. 添加新的适配器

**位置**：`chushihua_nonebot` 函数

**步骤**：
1. 导入新的适配器
2. 使用 `driver.register_adapter(新适配器)` 注册

## 6. 修改运行参数

**位置**：`nonebot.run()` 调用

**步骤**：
1. 修改 `host` 和 `port` 参数值
2. 可添加其他运行参数，如 `debug=True`

## 7. 添加新的消息发送功能

**位置**：可在 `bot.py` 中添加新的发送函数，或在插件中实现

**步骤**：
1. 定义新的异步发送函数
2. 使用 `bot.send_消息类型()` 方法发送消息
